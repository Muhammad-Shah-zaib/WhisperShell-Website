# WhisperShell Website

The official website for **[WhisperShell](https://github.com/Muhammad-Shah-zaib/WhisperShell)**, a high-performance, secure, Wayland-native offline voice-to-text dictation overlay for Linux.

## Purpose

This repository contains the source code for the WhisperShell website. Its primary purpose is to provide a central hub for users to:
- **Discover** the WhisperShell open-source project.
- **Download** the application for their Linux distributions (e.g., via RPM or Fedora COPR).
- **Read Documentation** covering everything from getting started and configuring AI models, to setting up global hotkeys and troubleshooting Wayland issues.

## Tech Stack

Built with modern web technologies:
- [Vue 3](https://vuejs.org/) (Composition API & `<script setup>`)
- [Vite](https://vitejs.dev/) (Build Tool)
- Vue Router (SPA Navigation)

## Local Development

To run the website locally for development or testing:

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start the development server:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```
